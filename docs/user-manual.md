# User Manual Streakly

Dokumen ini ditujukan untuk pengguna pemula agar bisa memakai Streakly dengan mudah dan aman.

## 1. Memulai Aplikasi
- Buka aplikasi Streakly dari desktop.
- Di halaman awal, Anda akan melihat dashboard ringkasan aktivitas.

## 2. Navigasi Utama
- **Dashboard**: Ringkasan tugas hari ini, XP, streak, dan insight singkat.
- **Calendar**: Melihat jadwal dan deadline berdasarkan tanggal.
- **Analytics**: Tren produktivitas 30 hari terakhir.
- **Task List**: Tambah, edit, dan kelola tugas secara detail.
- **Tracking**: Panel statistik fokus dan ritme kerja.
- **Setting**: Ubah tampilan, profil, dan pengaturan data.

## 3. Menambahkan Task
1. Masuk ke menu **Task List**.
2. Isi judul task, deadline, prioritas, status, dan assignee.
3. Klik **Add** untuk menyimpan.
4. Setelah tersimpan, task muncul di daftar dan dashboard.

## 4. Mengedit atau Menghapus Task
- Klik **Edit** pada task untuk memperbarui data.
- Klik **Hapus** untuk menghapus task yang tidak dibutuhkan.
- Klik **Done** untuk menandai task selesai dan menambah XP.

## 5. Kalender & Reminder
- Buka menu **Calendar** untuk melihat task berdasarkan tanggal.
- Klik tanggal untuk menambahkan task cepat (Quick Add).
- Atur reminder dari panel kalender untuk notifikasi fokus.

## 6. Analytics
- Lihat grafik XP 30 hari terakhir.
- Cek ringkasan streak dan konsistensi aktivitas.
- Gunakan insight untuk menjaga ritme produktif.

## 7. Backup, Restore, dan Export
Di **Setting**:
- **Backup Data**: Simpan database ke file lokal.
- **Restore Data**: Kembalikan data dari file backup.
- **Export PDF**: Buat laporan ringkas produktivitas.
- **Export Data (JSON)**: Unduh data lengkap untuk kebutuhan pribadi atau GDPR.

## 8. Privasi & Feedback
- Aktifkan **Analitik lokal** jika ingin menyimpan riwayat penggunaan di perangkat.
- Kirim masukan melalui kolom **Masukan pengguna** agar produk lebih baik.
