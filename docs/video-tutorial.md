# Video Tutorial Streakly (Naskah & Alur)

Dokumen ini adalah naskah dan alur pembuatan video tutorial singkat (5–7 menit).

## 1. Intro (0:00–0:30)
- Buka aplikasi Streakly.
- Jelaskan tujuan: membantu fokus, menjaga streak, dan melihat progres.

## 2. Navigasi Utama (0:30–1:30)
- Tunjukkan sidebar: Dashboard, Calendar, Analytics, Task List, Tracking, Setting.
- Jelaskan fungsi singkat tiap menu.

## 3. Tambah Task (1:30–3:00)
- Buka Task List.
- Isi judul, deadline, prioritas, status, dan assignee.
- Klik Add dan lihat task muncul di daftar.
- Tandai task selesai untuk menambah XP.

## 4. Kalender & Reminder (3:00–4:00)
- Buka Calendar.
- Tambah task cepat lewat tanggal.
- Atur reminder dan jelaskan notifikasi fokus.

## 5. Analytics (4:00–5:00)
- Buka Analytics.
- Tunjukkan grafik XP 30 hari.
- Jelaskan insight streak dan konsistensi.

## 6. Backup & Export (5:00–6:00)
- Masuk ke Settings.
- Demo Backup Data, Restore Data, Export PDF, Export Data (JSON).

## 7. Feedback & Privasi (6:00–7:00)
- Tunjukkan opsi Analitik lokal.
- Kirim masukan pengguna singkat.

## Penutup
- Ringkas manfaat: fokus, ritme, dan progres yang terukur.
- Ajakan: “Mulai satu task hari ini.”
